Содержание внутри страницы

Каждая страница отображает оглавление в правом верхнем углу. Но выводить содержание можно и непосредственно внутри страницы благодаря MDX.

Полноценное оглавление​

Переменная toc доступна на любой MDX странице и содержит все заголовки этой страницы. По умолчанию в оглавлении отображаются только заголовки h2 и h3. Вы можете управлять уровнями заголовков, установив minHeadingLevel или maxHeadingLevel для отдельных компонентов TOCInline.

import TOCInline from '@theme/TOCInline';

<TOCInline toc={toc} />
http://localhost:3000

Собственное оглавление​

Свойство toc — это просто список заголовков:

type TOCItem = {
  value: string;
  id: string;
  level: number;
};

Обратите внимание, что глобальный toc представляет собой плоский массив, поэтому вы можете легко вырезать ненужные узлы или вставить дополнительные узлы и создать новое дерево содержания.

import TOCInline from '@theme/TOCInline';

<TOCInline
  // Only show h2 and h4 headings
  toc={toc.filter((node) => node.level === 2 || node.level === 4)}
  minHeadingLevel={2}
  // Показывать заголовки h4 в дополнение к заголовкам h2 и h3
  maxHeadingLevel={4}
/>
http://localhost:3000
