# Песочница

Песочницы позволяют вам запускать Docusaurus **в вашем браузере, ничего не устанавливая**!

В основном они полезны для:

- Тестирования Docusaurus
- Reporting bugs

Используйте [docusaurus.new](https://docusaurus.new) в качестве простой для запоминания ссылки на песочницу.

Выберите один из доступных вариантов ниже.

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

Для удобства мы запомним ваш выбор при следующем посещении [docusaurus.new](https://docusaurus.new).

:::
