## Заголовки {#markdown-headings}

Вы можете использовать обычные Markdown заголовки.

```
## Заголовок 2-го уровня

### Заголовок 3-го уровня

#### Заголовок 4-го уровня
```

Такие заголовки выводятся в оглавлении страницы.

## Идентификаторы заголовков {#heading-ids}

Каждый заголовок имеет id, который может быть сгенерирован автоматически или явно указан.

Идентификаторы заголовков позволяют вам ссылаться на определенный заголовок страницы в Markdown или JSX:

```md
[ссылка](#heading-id)
```

```jsx
<Link to="#heading-id">ссылка</Link>
```

### Генерируемые идентификаторы {#generated-ids}

По умолчанию Docusaurus генерирует для вас идентификаторы заголовков на основе текста заголовка. Например, `### Hello World` будет иметь идентификатор `hello-world`.

Сгенерированные идентификаторы имеют **некоторые ограничения**:

- Id может выглядеть не очень хорошо
- Вы можете **изменить или перевести** текст без обновления существующего идентификатора

### Явные идентификаторы {#explicit-ids}

Специальный синтаксис Markdown позволяет задать **явный идентификатор заголовка**:

```md
### Hello World {#my-explicit-id}
```

:::tip

Воспользуйтесь CLI командой **[write-heading-ids](../../cli.md#docusaurus-write-heading-ids-sitedir)**, чтобы добавить явные идентификаторы ко всем вашим Markdown страницам.

:::
